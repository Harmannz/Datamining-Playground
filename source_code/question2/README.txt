BPNN:
BPNN directory contains BPNN library with the network file and patter file for XOR problem.
To run:
1. remove the nntrain and nntest file if they exist
2. run make
3. Train the neural network by `./nntrain digits.net digits05.pat`
4. Test the neural networb by `./nntest xorplus.net digits05.pat weights.dat`

Other patters such as digits15.pat, digits30.pat and digits50.par can also be tested similarly
